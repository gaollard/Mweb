# 2.1 Function 函数

## 属性

* function.length
形参的个数

* function.caller(非标准)
它指向调用当前函数的函数，如果函数是在全局范围内调用的话，那么caller的值为null。

## 类方法

* Array.isArray()
判断是否为数组

* Array.from()
将类数组转换为数组

## 原型方法

* Function.prototype.constructor()
* Function.prototype.bind()
* Function.prototype.apply()
* Function.prototype.call()
